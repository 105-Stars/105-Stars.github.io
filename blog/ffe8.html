提高Github访问速度

一键修改Hosts文件

Github 在全球各地都有服务器，而国内 DNS 解析服务基本上都是把域名解析到了美国的服务器，所以访问起来会比较慢，但是，我们可以通过修改本地 hosts 文件来绕过国内 DNS 解析，直接访问 Github 在韩国、日本等的服务器（通常访问这俩国的服务器会比较快），从而达到加速访问的目的。

推荐

个人推荐使用由羽翼城大佬开发的知名项目UsbEAm Hosts Editor进行一键修改hosts文件。因为这个软件的使用方法简单，同时适合新手使用。软件可以直接到大佬博客上进行下载或者使用我分享的地址下载，下载地址在文章结尾。

使用方法

1、打开软件，点击软件左下角，选择准备修改 hosts 的网站

2、检测延迟 » 选择延迟最低的IP地址 » 应用选中

3、打开浏览器 » 访问Github

其他功能

这款软件还可以用来加速 Steam、暴雪、育碧、Microsoft Store 等平台的访问与下载！（更多有趣功能等待大家的探索！）
